# Scrapers

Personal scrapers for study notes and data collection.

- [`article-to-markdown/`](./article-to-markdown) — convert any article URL to local markdown with embedded images

## Conventions

- Each tool is self-contained with its own README and requirements.txt
- Each tool runs in its own venv (`cd <tool> && python -m venv .venv`)
- Output goes to gitignored folders by default
